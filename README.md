# Shape Generator - Assembly & C#

This project is a university assignment that generates different geometric shapes using:

- Assembly Language (NASM)
- C#

## Shapes Generated:
- Circle
- Square
- Triangle
- Rectangle
- Diamond

## Features:
- User input based shape size
- Console-based shape rendering
- Multiple shape selection menu
- Implemented in both Assembly and C#

## Project Structure:
- /Assembly → Assembly implementation (.asm files)
- /CSharp → C# implementation (.cs files)

## Technologies Used:
- NASM Assembly
- C#
- Console Output
